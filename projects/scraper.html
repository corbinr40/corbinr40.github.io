PlayStation Xbox Scraper

A Python script using the Selenium library to scrape the Xbox and the PlayStation stores for the Let’s Play app.

Overview

Project Status: Ongoing
Project Type: Personal/Script
Project Duration: ~2 months
Software Used: Python IDE, Selenium
Languages Used: Python
Available On: GitHub

Features

A custom written scraper using Python and the Selenium framework to gather data from the Xbox and PlayStation store for my “Let’s Play” (link) program.

This CLI program can be run on multiple platforms (tried on MacOS, Windows 10 and Ubuntu) which can require different browsers (Safari, Chrome, Edge, Firefox, etc.).

Saves data to a CSV file and converted to a custom written TypeScript database file
