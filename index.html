Welcome!

Hi, I’m Corbin Richardson, and welcome to my site where I show off what I've done and what I'm currently doing. I’m a 1st Class Computer Science MEng Graduate from Leeds Beckett University. I can develop for desktop, mobile (using Mutli-Platform Frameworks), web design, and game development/design, whilst 3D modelling as a hobby. Regarding software development, I am proficient in Python, C#, Java, and Web Development (HTML, CSS, JavaScript). But, I’m always wanting to learn way more! For game dev/design I use both Unreal Engine and Unity. Unreal Engine is mainly used for personal projects and to show off some 3D models. I use Unity for nearly all game jams I participate in. When it comes to modelling there are many different aspects. To model I use Blender, for texturing, I use Adobe’s Substance Suite (Substance Painter, Substance Designer and Substance Sampler) and Krita.
